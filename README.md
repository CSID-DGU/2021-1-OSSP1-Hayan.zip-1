### 소개
발표나 면접을 위한 암기를 보조하는 프로그램이다.
사용자에게 대본을 입력받은 후 음성인식을 진행한다. 대본과 음성을 특정 조건에 맞추어 비교하여 일치여부 판단하는 프로그램이다.
